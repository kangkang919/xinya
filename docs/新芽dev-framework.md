| 场景 | 要求 |
|---|---|

| 数据库连接失败 | 启动时报错，不静默忽略 |
| 前端 API 错误 | Toast 提示，不 alert |

### 5.4 各服务模块

#### 服务 1: 心得服务

- **职责：** 心得的 CRUD、搜索、筛选、置顶/收藏切换
- **输入：** 用户 ID、心得数据、筛选参数
- **处理规则：**
  1. 保存时标题非空
  2. 无标签时自动补"随笔"
  3. `isDraft` 固定为 `false`
- **异常处理：** 标题为空返回 400；未找到返回 404

#### 服务 2: 标签服务

- **职责：** 标签 CRUD、删除时补默认标签
- **输入：** 用户 ID、标签 ID/名称
- **处理规则：**
  1. 创建时校验名称长度 ≤8
  2. 删除前给仅含该标签的心得补"随笔"
  3. 禁止删除默认"随笔"标签

#### 服务 3: 统计服务

- **职责：** 今日、本周、连续、最长连续统计
- **输入：** 用户 ID
- **处理规则：**
  1. 所有时间按北京时间计算
  2. 周一为周起点
  3. 连续天数从今天或昨天开始往前数

### 5.5 API 路由

```
GET    /api/entries?search=&favorite=&tagId=&from=&to=&page=&limit=
POST   /api/entries
  - 输入：{ title, content, mood, tagIds }
  - 规则：无 tagIds 自动补默认标签；isDraft=false
GET    /api/entries/[id]
PUT    /api/entries/[id]
  - 输入：{ title, content, mood, tagIds }
  - 规则：无 tagIds 自动补默认标签
DELETE /api/entries/[id]
PATCH  /api/entries/[id]
  - 输入：{ isTop } 或 { isFavorite }
```

```

```

```
  - 输出：{ todayCount, weekCount, streak, maxStreak, lastEntry }
```
