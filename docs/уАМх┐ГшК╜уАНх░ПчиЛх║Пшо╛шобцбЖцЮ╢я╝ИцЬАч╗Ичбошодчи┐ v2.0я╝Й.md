> **版本**：v2.0

> **状态**：已确认

## 一、产品定位

| 项目 | 内容 |
| :--- | :--- |
| **名称** | 心芽 |
| **Slogan** | 记录内心的每一次萌发 |
| **核心价值** | 私有记录 + 可控分享 |
| **技术方案** | 纯响应式网站：PC端独立网页，手机端自动适配，支持"添加到桌面" |
| **双端策略** | 网页端专注写作编辑，手机端便捷回顾学习；手机端与网页端富文本编辑功能完全一致 |

---

## 二、四页面架构

| 页面 | 功能定位 | Slogan |
| :--- | :--- | :--- |
| 萌芽 | 心得创建与时间轴列表 | 每一颗灵感的种子，都在此刻破土而出，迎接第一缕晨光，请在此播下你的思绪 |
| 枝叶 | 按标签维度组织浏览 | 繁茂的枝叶向四面八方舒展，轻触标签的脉络，便能寻回那片承载记忆的叶子 |
| 年轮 | 数据统计与心理成长分析 | 时光在树干上刻下深浅不一的纹路，待岁月沉淀，此处将映照出你成长的轨迹 |
| 根系 | 基础设置/分享/主题/版本 | 根系在泥土深处悄然延伸，在此安顿偏好，静待功能丰盈时托起整片森林 |

## 三、心得数据模型

| 字段 | 规则 |
| :--- | :--- |
| 标题 | 必填，简短 |
| 正文 | 富文本编辑器（见第四章） |
| 标签 | 至少选择一个；纯文字，≤8字；支持多标签；未选择时自动使用默认标签"随笔" |
| 心情标记 | 可选，5个选项：开心 😊、难过 😢、平静 😐、激动 🤩、忧虑 😟 |
| 记录时间 | 自动生成 + 可手动修改 |
| 收藏状态 | 可选，默认未收藏；卡片右上角书签图标操作 |

> **已删除字段**：配图（图片功能整体删除）

## 四、富文本编辑器规范

### 4.1 通用规则（手机端与网页端完全一致）

| 项目 | 规则 |
| :--- | :--- |
| 默认字体 | 微软雅黑 |
| 默认字号 | 12号 |
| 字体颜色 | 仅四色可选——红、黄、蓝、默认黑色 |
| 无标题层级 | 不设H1/H2/H3等标题选择 |
| 粘贴净化 | 所有外部粘贴文本统一强制转化为上述格式；不支持图片粘贴或上传 |
| 支持功能 | 加粗、斜体、有序/无序列表、撤销/重做 |
| 自动保存 | 每30秒自动保存草稿 |
| 字数统计 | 实时显示在编辑器右下角，浅灰色 |

### 4.2 网页端专属增强

- 专注模式（全屏编辑，按ESC或左上角按钮退出）

### 4.3 离线状态处理

用户在无网络时点击发布，弹出提示：

> **"种子还未落地，信号不在身边 🌿"**
> 你的心得已保存为草稿，待信号回来再发布吧。

> **已删除功能**：插入图片、历史版本回溯、手机端差异化功能说明

## 五、交互设计

### 5.1 全局交互

| 项目 | 规则 |
| :--- | :--- |
| 底部导航 | 四Tab导航栏：萌芽、枝叶、年轮、根系；中央常驻"+"新建按钮，点击直接进入编辑 |
| 底部导航安全边距 | 自动适配手机系统底部区域（Safe Area），确保所有机型按钮不被遮挡 |
| 全局搜索 | 仅在萌芽页和枝叶页显示，年轮页与根系页不显示 |
| 过渡动画 | 页面切换加200-300ms轻量过渡动画 |
| 多设备登录 | 允许多设备同时登录，不互相踢下线 |

### 5.2 萌芽页

**页面结构（从上到下）：**
1. 顶部固定导航栏（页面标题 + 布局切换 + 退出按钮）
2. 搜索栏 + 收藏筛选按钮（⭐）+ 时间范围快捷筛选（最近7天 / 30天 / 3个月 / 全部）
3. 今日速览卡片（随内容一起滚动）
4. 心得卡片列表

**今日速览卡片规则：**
- 展示：今日是否已记录、本周记录频率、最近一条心得预览、连续记录天数
- 今日已记录≥1条后，自动折叠为一行小字"今日已播种 🌱"，点击可展开

**卡片展示规则：**

| 项目 | 规则 |
| :--- | :--- |
| 布局 | 默认卡片流，支持切换时间轴布局 |
| 卡片内容 | 标题、首行预览（2行截断）、标签胶囊、记录时间、心情图标 |
| 收藏图标 | 卡片右上角书签形状；未收藏灰色空心，已收藏嫩绿色实心；点击触发弹跳动画 |
| 长按操作 | 编辑、删除、置顶 |
| 置顶规则 | 最多1篇置顶 |
| 收藏规则 | 多篇可同时收藏 |
| 收藏筛选 | 搜索栏旁⭐按钮，点击只显示收藏心得，再点取消 |
| 新建引导文案 | "此刻思绪万千，别忘了为它贴上专属的叶片标签 🌿" |

**删除操作弹窗文案：**

> **"确定要让这片叶子飘落吗？"**
> 一旦飘落，便无法追回。
>
> 【让它飘落】　　　【再想想】

确认删除后直接删除，无回收站。

### 5.3 枝叶页

| 项目 | 规则 |
| :--- | :--- |
| 搜索框功能 | 搜索标签名称，输入关键词后高亮匹配的标签气泡 |
| 第一级 | 标签云/网格，气泡大小分3级：小1-5篇、中6-15篇、大16篇以上，瀑布流排列 |
| "随笔"标签 | 固定置底显示，不参与气泡大小排序 |
| 第二级 | 点击标签展开该标签下心得列表（支持时间排序） |
| 标签管理 | 自定义创建 + 系统推荐标签；纯文字≤8字；每用户最多20个标签；支持改名（自动更新关联心得）和删除（只移除该标签，心得的其他标签保留；若该心得无其他标签则自动归入"随笔"） |

### 5.4 年轮页

| 项目 | 规则 |
| :--- | :--- |
| 日历热力图 | 类GitHub贡献图，展示每日记录频率 |
| 趋势图 | 周/月趋势折线图 |
| 连续记录 | 从首次记录算起，断开保留历史最长记录（格式：已连续X天 / 历史最长Y天） |
| 洞察报告生成时机 | 每周一早6点生成上周报告；每月1日早6点生成上月报告 |
| 洞察报告查看入口 | 在根系页查看 |
| AI洞察触发规则 | 每记录5的倍数次（5/10/15...）自动触发一次AI分析 |
| AI洞察内容 | 分析用户记录习惯和特征，使用硅基流动免费模型；效果满意后可升级付费模型 |
| AI洞察展示位置 | 年轮页底部固定"心芽洞察卡片"，卷轴展开样式，每次更新显示"新的洞察已生长 🌱"小动画 |
| AI洞察空状态文案 | "你的第一份洞察正在孕育，再记录几次，心芽就会读懂你 🌱" |
| 年轮空状态文案 | "时光在树干上刻下深浅不一的纹路，待岁月沉淀，此处将映照出你成长的轨迹。" |

### 5.5 根系页

| 项目 | 规则 |
| :--- | :--- |
| 分享管理 | 分享链接列表（每条显示：创建时间、有效期剩余天数、分享范围、一键复制、一键撤销）；支持创建多个链接；有效期选项：7天/30天/90天/自定义；可见范围：全部或特定标签；访客只读，界面标注"只读模式"，访客使用默认主题；底部引导：「觉得好用？创建自己的心芽账户 →」 |
| 背景风格 | 5套主题：春日萌芽（默认）、夏日繁茂、秋日暖阳、冬日静谧、夜间模式 |
| 数据导出 | 导出为系统最方便处理的格式（JSON/Markdown），附带说明文档 |
| 成长日志 | 大版本迭代时间轴，由开发者手动维护；初始预填：「2026年，心芽破土而出，迎来第一缕阳光 🌱」；后续文案风格与产品语境保持一致 |
| 洞察报告 | 每周/每月报告在此查看 |
| 连续记录天数 | 打卡激励展示 |
| 问题反馈 | 邮箱：1243177461@qq.com |
| 离线缓存设置 | 显示当前缓存状态（最近50篇或最近1个月） |
| 添加桌面引导 | 第2-3次打开时提示；文案："想让心芽像APP一样随时打开？👇 点击浏览器右上角「...」→ 添加到桌面，你的心芽花园，触手可及 🌱" |

## 七、新用户引导（播种引导）

**触发时机**：首次注册成功后自动进入

**三步流程：**
1. 欢迎插画页（嫩芽破土）+ 欢迎语：「欢迎来到心芽，这里是你内心的花园 🌱」
2. 创建第一个标签引导
3. 写下第一条心得引导

**每步都有"跳过，直接进入"小字按钮**，点击后直接进入主界面。

## 八、视觉与品牌风格

### 8.1 色彩体系

| 颜色 | 用途 |
| :--- | :--- |
| 嫩绿色系（#8BC34A / #AED581） | 主色调 |
| 暖白底色（#FAFAF5） | 背景 |
| 大地棕 #795548 | 标签/强调 |
| 天空蓝 #42A5F5 | 图表数据 |
| 字体 | 圆体中文字体（如思源圆体） |

### 8.2 背景主题（5套）

| 主题 | 色系 |
| :--- | :--- |
| 春日萌芽（默认） | 浅绿暖白 |
| 夏日繁茂 | 深绿色系 |
| 秋日暖阳 | 橙黄暖色调 |
| 冬日静谧 | 灰白极简 |
| 夜间模式 | 深色护眼 |

### 8.3 插图与动画风格

线条手绘风，叶子、枝丫、年轮纹、根须等自然形态，铅笔手绘质感，温暖有机。四个页面空状态各一张插画，带微动画，风格统一。

**加载状态**：嫩芽生长动画代替传统转圈loading。

### 8.4 核心文案规范

**操作反馈文案：**
- 发布成功：「一颗新芽破土而出 🌱」
- 删除确认：「确定要让这片叶子飘落吗？一旦飘落，便无法追回。」
- 按钮：【让它飘落】【再想想】
- 分享成功：「一粒种子已随风飘向远方」
- 离线保存：「种子还未落地，信号不在身边 🌿 你的心得已保存为草稿，待信号回来再发布吧。」
