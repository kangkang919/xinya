# 心芽项目资料索引

> 本文档是心芽项目的总览索引，帮助新成员或换电脑后快速了解项目全貌。

## 项目一句话定位

**心芽** — 面向个人的私有化心得记录应用，四页面架构（萌芽/枝叶/年轮/根系），支持富文本记录、标签组织、数据统计、AI 洞察、截图分享。

## 开发工作流

```
本地写代码 → git push GitHub → 服务器 git pull → npm run build → pm2 restart
```

- **本地**：只写代码和调试，不部署
- **GitHub**：代码唯一来源，`kangkang919/xinya`
- **服务器**：`/www/wwwroot/xinya`，PM2 管理进程
