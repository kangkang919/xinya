# 邮箱登录系统方案介绍

> 本文档详细介绍 Next.js 项目中邮箱登录系统的完整实现方案，包括邮箱验证登录、邀请码登录、邮箱+密码双模式登录的具体操作步骤和代码参考。
> 适用于需要实现多种邮箱登录方式的 Web 应用项目。

---

## 一、方案概览

本项目的邮箱登录系统包含三种登录方式：

| 登录方式 | 适用场景 | 用户体验 |
|---------|---------|---------|
| **邮箱 + 密码登录** | 常规用户登录 | 输入邮箱和密码，点击登录 |
| **Magic Link 邮箱链接登录** | 无密码登录、跨设备登录 | 输入邮箱，收邮件，点击链接自动登录 |
| **邀请码登录** | 内测/邀请制用户 | 输入预生成的账号密码，一键登录 |

---

## 二、邮箱 + 密码登录

### 2.1 数据库设计

```prisma
model User {
  id        Int      @id @default(autoincrement())
  email     String   @unique
  password  String   // bcrypt 哈希后的密码
  theme     String   @default("spring")
  openTimes Int      @default(0)
  createdAt DateTime @default(now())
  updatedAt DateTime @updatedAt
  
  entries   Entry[]
}
```

### 2.2 注册流程

```typescript
// app/api/auth/register/route.ts
import { PrismaClient } from '@prisma/client'
import bcrypt from 'bcryptjs'
import { SignJWT } from 'jose'

const prisma = new PrismaClient()
const JWT_SECRET = new TextEncoder().encode(process.env.JWT_SECRET!)

export async function POST(request: Request) {
  const { email, password } = await request.json()
  
  // 1. 检查邮箱是否已注册
  const existing = await prisma.user.findUnique({ where: { email } })
  if (existing) {
    return Response.json({ ok: false, error: '邮箱已被注册' }, { status: 400 })
  }
  
  // 2. 密码加密（bcrypt，salt rounds = 10）
  const hashedPassword = await bcrypt.hash(password, 10)
  
  // 3. 创建用户
  const user = await prisma.user.create({
    data: { email, password: hashedPassword }
  })
  
  // 4. 生成 JWT Token
  const token = await new SignJWT({ userId: user.id })
    .setProtectedHeader({ alg: 'HS256' })
    .setExpirationTime('7d')
    .sign(JWT_SECRET)
  
  // 5. 设置 HttpOnly Cookie
  const response = new Response(JSON.stringify({ ok: true }))
  response.headers.set('Set-Cookie', 
    `xinya-token=${token}; HttpOnly; Path=/; Max-Age=${7 * 24 * 60 * 60}; SameSite=Lax`
  )
  
  return response
}
```

### 2.3 登录流程

```typescript
// app/api/auth/login/route.ts
import { PrismaClient } from '@prisma/client'
import bcrypt from 'bcryptjs'
import { SignJWT } from 'jose'

const prisma = new PrismaClient()
const JWT_SECRET = new TextEncoder().encode(process.env.JWT_SECRET!)

export async function POST(request: Request) {
  const { email, password } = await request.json()
  
  // 1. 查找用户
  const user = await prisma.user.findUnique({ where: { email } })
  if (!user) {
    return Response.json({ ok: false, error: '邮箱或密码错误' }, { status: 401 })
  }
  
  // 2. 验证密码
  const valid = await bcrypt.compare(password, user.password)
  if (!valid) {
    return Response.json({ ok: false, error: '邮箱或密码错误' }, { status: 401 })
  }
  
  // 3. 更新打开次数
  await prisma.user.update({
    where: { id: user.id },
    data: { openTimes: { increment: 1 } }
  })
  
  // 4. 生成 JWT Token
  const token = await new SignJWT({ userId: user.id })
    .setProtectedHeader({ alg: 'HS256' })
    .setExpirationTime('7d')
    .sign(JWT_SECRET)
  
  // 5. 设置 Cookie（包含主题信息）
  const response = new Response(JSON.stringify({ 
    ok: true, 
    data: { email: user.email, theme: user.theme }
  }))
  response.headers.set('Set-Cookie', 
    `xinya-token=${token}; HttpOnly; Path=/; Max-Age=${7 * 24 * 60 * 60}; SameSite=Lax`
  )
  
  return response
}
```

### 2.4 前端登录表单

```tsx
// app/(auth)/login/page.tsx
"use client"
import { useState } from "react"
import { useRouter } from "next/navigation"

export default function LoginPage() {
  const router = useRouter()
  const [email, setEmail] = useState("")
  const [password, setPassword] = useState("")
  const [error, setError] = useState("")
  const [loading, setLoading] = useState(false)

  async function handleLogin() {
    setError("")
    setLoading(true)
    
    try {
      const res = await fetch("/api/auth/login", {
        method: "POST",
        headers: { "Content-Type": "application/json" },
        body: JSON.stringify({ email, password }),
      })
      const json = await res.json()
      
      if (!json.ok) {
        setError(json.error)
        return
      }
      
      // 保存主题到 localStorage
      if (json.data?.theme) {
        localStorage.setItem("xinya-theme", json.data.theme)
      }
      
      router.push("/")
    } catch (e) {
      setError("网络错误，请重试")
    } finally {
      setLoading(false)
    }
  }

  return (
    <div className="min-h-screen flex items-center justify-center p-4">
      <div className="w-full max-w-sm">
        <h1 className="text-2xl font-bold text-center mb-8">心芽</h1>
        
        {error && (
          <p className="text-sm text-red-500 mb-4 text-center">{error}</p>
        )}
        
        <input
          type="email"
          placeholder="邮箱"
          value={email}
          onChange={e => setEmail(e.target.value)}
          className="w-full px-4 py-3 rounded-xl border mb-3"
        />
        
        <input
          type="password"
          placeholder="密码"
          value={password}
          onChange={e => setPassword(e.target.value)}
          onKeyDown={e => e.key === "Enter" && handleLogin()}
          className="w-full px-4 py-3 rounded-xl border mb-4"
        />
        
        <button
          onClick={handleLogin}
          disabled={loading}
          className="w-full py-3 rounded-xl bg-green-500 text-white font-medium"
        >
          {loading ? "登录中…" : "登录"}
        </button>
      </div>
    </div>
  )
}
```

## 三、Magic Link 邮箱链接登录

### 3.1 核心流程

```
用户输入邮箱 → 后端生成 token → 发送含 token 的邮件 → 用户点击邮件链接 → 自动登录
```

### 3.2 数据库设计

```prisma
model MagicLink {
  id        Int      @id @default(autoincrement())
  email     String
  token     String   @unique
  expiresAt DateTime
  used      Boolean  @default(false)
  createdAt DateTime @default(now())
}
```

### 3.3 发送 Magic Link

```typescript
// app/api/auth/magic-link/send/route.ts
import { PrismaClient } from '@prisma/client'
import { randomBytes } from 'crypto'
import nodemailer from 'nodemailer'

const prisma = new PrismaClient()

// 邮件 transporter 配置（以 QQ 邮箱为例）
const transporter = nodemailer.createTransport({
  host: 'smtp.qq.com',
  port: 465,
  secure: true,
  auth: {
    user: process.env.SMTP_USER,
    pass: process.env.SMTP_PASS,
  },
})

export async function POST(request: Request) {
  const { email } = await request.json()
  
  // 1. 生成随机 token（32 字节，hex 编码）
  const token = randomBytes(32).toString('hex')
  
  // 2. 设置过期时间（15 分钟后）
  const expiresAt = new Date(Date.now() + 15 * 60 * 1000)
  
  // 3. 保存到数据库
  await prisma.magicLink.create({
    data: { email, token, expiresAt }
  })
  
  // 4. 生成登录链接
  const loginUrl = `${process.env.NEXT_PUBLIC_APP_URL}/api/auth/magic-link/verify?token=${token}`
  
  // 5. 发送邮件
  await transporter.sendMail({
    from: `"心芽" <${process.env.SMTP_USER}>`,
    to: email,
    subject: '心芽登录链接',
    html: `
      <p>点击以下链接登录心芽：</p>
      <a href="${loginUrl}">${loginUrl}</a>
      <p>链接 15 分钟内有效，且只能使用一次。</p>
      <p>如果不是你本人操作，请忽略此邮件。</p>
    `,
  })
  
  return Response.json({ ok: true })
}
```

### 3.4 验证 Magic Link

```typescript
// app/api/auth/magic-link/verify/route.ts
import { PrismaClient } from '@prisma/client'
import { SignJWT } from 'jose'
import { cookies } from 'next/headers'

const prisma = new PrismaClient()
const JWT_SECRET = new TextEncoder().encode(process.env.JWT_SECRET!)

export async function GET(request: Request) {
  const { searchParams } = new URL(request.url)
  const token = searchParams.get('token')
  
  if (!token) {
    return Response.redirect(`${process.env.NEXT_PUBLIC_APP_URL}/login?error=无效链接`)
  }
  
  // 1. 查找 token 记录
  const record = await prisma.magicLink.findUnique({ where: { token } })
  
  if (!record || record.used) {
    return Response.redirect(`${process.env.NEXT_PUBLIC_APP_URL}/login?error=链接已失效`)
  }
  
  // 2. 检查是否过期
  if (new Date() > record.expiresAt) {
    return Response.redirect(`${process.env.NEXT_PUBLIC_APP_URL}/login?error=链接已过期`)
  }
  
  // 3. 查找或创建用户
  let user = await prisma.user.findUnique({ where: { email: record.email } })
  if (!user) {
    user = await prisma.user.create({
      data: { email: record.email }
    })
  }
  
  // 4. 标记 token 已使用（一次性）
  await prisma.magicLink.update({
    where: { id: record.id },
    data: { used: true }
  })
  
  // 5. 生成 JWT Token
  const jwt = await new SignJWT({ userId: user.id })
    .setProtectedHeader({ alg: 'HS256' })
    .setExpirationTime('7d')
    .sign(JWT_SECRET)
  
  // 6. 设置 Cookie 并重定向到首页
  const response = new Response(null, {
    status: 302,
    headers: {
      'Location': process.env.NEXT_PUBLIC_APP_URL!,
      'Set-Cookie': `xinya-token=${jwt}; HttpOnly; Path=/; Max-Age=${7 * 24 * 60 * 60}; SameSite=Lax`
    },
  })
  
  return response
}
```

## 四、邀请码登录

### 4.1 预生成账号

```typescript
// scripts/generate-invite-accounts.ts
import { PrismaClient } from '@prisma/client'
import bcrypt from 'bcryptjs'
import { randomBytes } from 'crypto'

const prisma = new PrismaClient()

async function main() {
  const accounts = []
  
  for (let i = 1; i <= 15; i++) {
    const email = `guest${String(i).padStart(2, '0')}@shuxiangnote.top`
    const password = randomBytes(4).toString('hex') // 8 位随机密码
    const hashedPassword = await bcrypt.hash(password, 10)
    
    await prisma.user.create({
      data: { email, password: hashedPassword }
    })
    
    accounts.push({ email, password })
  }
  
  console.log('生成的账号：')
  console.table(accounts)
  
  // 输出为 Markdown 文档
  const md = accounts.map(a => `| ${a.email} | ${a.password} |`).join('\n')
  require('fs').writeFileSync('邀请码.md', 
    `# 心芽邀请账号\n\n| 邮箱 | 密码 |\n|------|------|\n${md}\n`
  )
}

main().catch(console.error)
```

### 4.2 邀请码登录流程

前端可以添加一个「邀请码登录」入口，点击后显示账号密码列表供用户复制：

## 五、从邮箱登录升级为邮箱+密码双模式

### 5.1 场景说明

项目初期可能只实现了 Magic Link 邮箱链接登录（无密码），后续需要支持邮箱+密码登录。

### 5.2 升级步骤

#### 步骤 1：数据库添加 password 字段

```prisma
model User {
  id        Int      @id @default(autoincrement())
  email     String   @unique
  password  String?  // ← 改为可选，Magic Link 用户初始无密码
  theme     String   @default("spring")
  // ...
}
```

执行迁移：
```bash
npx prisma migrate dev --name add-password-field
```

#### 步骤 2：修改登录 API 支持两种模式

```typescript
// app/api/auth/login/route.ts
export async function POST(request: Request) {
  const { email, password, useMagicLink } = await request.json()
  
  const user = await prisma.user.findUnique({ where: { email } })
  if (!user) {
    return Response.json({ ok: false, error: '邮箱未注册' }, { status: 401 })
  }
  
  // 如果使用 Magic Link 模式
  if (useMagicLink) {
    // 发送 Magic Link 邮件（参考第三节）
    return sendMagicLink(email)
  }
  
  // 如果使用密码模式
  if (!user.password) {
    return Response.json({ 
      ok: false, 
      error: '该账号未设置密码，请使用邮箱链接登录' 
    }, { status: 401 })
  }
  
  const valid = await bcrypt.compare(password, user.password)
  if (!valid) {
    return Response.json({ ok: false, error: '密码错误' }, { status: 401 })
  }
  
  // 登录成功...
}
```

#### 步骤 3：添加设置密码功能

允许 Magic Link 用户后续设置密码：

```typescript
// app/api/auth/set-password/route.ts
import { PrismaClient } from '@prisma/client'
import bcrypt from 'bcryptjs'
import { cookies } from 'next/headers'
import { jwtVerify } from 'jose'

const prisma = new PrismaClient()
const JWT_SECRET = new TextEncoder().encode(process.env.JWT_SECRET!)

export async function POST(request: Request) {
  // 1. 验证当前登录状态
  const cookieStore = cookies()
  const token = cookieStore.get('xinya-token')?.value
  
  if (!token) {
    return Response.json({ ok: false, error: '未登录' }, { status: 401 })
  }
  
  const { payload } = await jwtVerify(token, JWT_SECRET)
  const userId = payload.userId as number
  
  // 2. 获取新密码
  const { password } = await request.json()
  if (!password || password.length < 6) {
    return Response.json({ ok: false, error: '密码至少6位' }, { status: 400 })
  }
  
  // 3. 加密并保存
  const hashedPassword = await bcrypt.hash(password, 10)
  await prisma.user.update({
    where: { id: userId },
    data: { password: hashedPassword }
  })
  
  return Response.json({ ok: true })
}
```

## 六、安全注意事项

### 6.1 密码安全
- 使用 bcrypt 加密，salt rounds ≥ 10
- 前端传输使用 HTTPS
- 不在日志中记录密码

### 6.2 JWT Token 安全
- Token 有效期建议 7 天
- 使用 HttpOnly Cookie 存储，防止 XSS 攻击
- Secret 密钥长度 ≥ 32 字符

### 6.3 Magic Link 安全
- Token 使用 crypto.randomBytes 生成，不可预测
- Token 一次性使用，用后立即标记
- 15 分钟过期时间
- 邮件中明确提示"如果不是你本人操作，请忽略"

### 6.4 邀请码安全
- 预生成账号的密码使用随机生成
- 邀请码文档妥善保管，不公开
- 可限制每个邀请码只能登录一次（可选）
