## 五、重要提醒

1. **`.env` 和 `.env.production` 包含敏感信息**，不要上传到 GitHub（已在 .gitignore 中排除）
2. **GitHub PAT 有有效期**，过期后需要重新生成
3. **QQ 邮箱 SMTP 授权码**如果长时间不用可能失效，需要重新生成
4. **数据库密码**如果不记得，在服务器上 `cat /www/wwwroot/xinya/.env.production | grep DATABASE_URL` 查看
5. **PowerShell 路径含空格**：`cd "D:\Project Qoder\xinya"` 必须用引号包裹路径
6. **双平台推送**：每次本地修改后需分别推送到 GitHub 和 Gitee

8. **本地开发直连服务器数据库**，不需要本地装 PostgreSQL
