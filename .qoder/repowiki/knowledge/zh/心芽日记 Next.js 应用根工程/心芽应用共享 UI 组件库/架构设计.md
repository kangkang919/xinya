本模块是纯客户端 React 组件集合，每个文件导出一个独立组件，无内部子目录或共享类型定义，依赖方向单向：组件 -> Next.js 路由 (next/navigation) -> /api/* REST 接口。
- Editor.tsx 作为编排器组合 EditorToolbar.tsx（工具栏）、内置 contentEditable 正文区、心情选择器和标签选择器，并通过 useTheme() 注入主题色；EditorToolbar.tsx 仅负责按钮与颜色拾取器，通过回调将 execCommand/insertList 等操作上抛。
- EntryCard.tsx 负责列表项展示与收藏/置顶/删除的乐观更新 + 回滚逻辑，通过 onToggleFavorite/onTogglePin/onDelete 回调将副作用上抛给父页面。
- DeleteDialog.tsx 是无状态确认弹窗，仅接收 open/onConfirm/onCancel/loading 控制。
- SharePanel.tsx 使用动态 script 加载 CDN 上的 html2canvas@1.4.1，将 DOM 渲染为图片后走 Web Share API 或降级下载。
- review-card.tsx 实现三态流程（正面概念名 -> 背面要点 -> 答题/反馈），提交到 /api/review/answer 并支持回看解析。
所有组件均标注 "use client"，样式以 Tailwind 类名为主、辅以内联 style 覆盖主题色，图标统一来自 lucide-react。
