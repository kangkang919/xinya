本模块为纯一次性/周期性运行的 Node.js 脚本集合，不暴露 HTTP 接口，也不被业务代码 import。
- 依赖方向：所有脚本均直接 `require` / `import @prisma/client` 操作数据库，绕过应用层服务，属于“直连 Prisma”的运维通道。
- 脚本职责划分：
  - `create-migration.js`：硬编码 SQL DDL，向 `prisma/migrations/20260621_init/` 写出 `migration.sql` 与 `migration_lock.toml`，用于首次建库。
  - `create-guest-accounts.js`：用 bcryptjs 哈希密码，批量创建 guest@shuxiangnote.top 用户并附带默认 Tag。
  - `backfill-questions.ts` / `regenerate-all-summaries.ts`：通过环境变量 `DEEPSEEK_API_KEY` + `DEEPSEEK_BASE_URL` 调用 DeepSeek `/chat/completions`，按固定 prompt 解析 JSON，回写 `entry.keyPoints` 并批量插入 `quizQuestion` 与 `quizRecord`；前者仅处理缺失字段，后者全量重建旧题。
- 运行入口：每个脚本以 `main().catch(...)` 自执行，结束时显式 `prisma.$disconnect()`，由外部通过 `npx tsx scripts/<script>.ts` 或 `node scripts/<script>.js` 触发。
