- 脚本以顶层 `main()` 函数组织主流程，并以 `main().catch(console.error)` 作为自执行入口，结束时统一调用 `prisma.$disconnect()`。
- 对外部 LLM 调用采用 AbortController + setTimeout 实现 30 秒超时保护，失败时返回空结果并走模板降级逻辑。
- 通过 `process.env.DEEPSEEK_API_KEY` / `DEEPSEEK_BASE_URL` 注入凭据，避免在脚本中硬编码密钥。
- 对 AI 返回的文本使用正则 `\{[\s\S]*\}` 提取 JSON 片段后再 `JSON.parse`，增强对非严格 JSON 响应的容错。
