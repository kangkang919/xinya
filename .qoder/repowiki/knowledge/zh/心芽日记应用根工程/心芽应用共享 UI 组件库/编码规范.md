- 组件顶部声明 "use client"，全部运行在浏览器端，通过 fetch('/api/...') 直接调用服务端 API。
- 对外暴露受控 props（如 open/onConfirm/onCancel、onToggleFavorite/onTogglePin/onDelete），副作用由父组件驱动，组件自身不持有业务数据。
- 主题色通过 isDark prop 或 useTheme() 获取，颜色值以内联 style 形式注入，而非 CSS 变量。
- 异步操作采用 try/catch + toast 反馈模式，失败时回滚本地 optimistic 状态（如 EntryCard 收藏按钮）。
- 第三方重型库（html2canvas）通过动态创建 script 按需加载，避免首屏打包体积膨胀。
